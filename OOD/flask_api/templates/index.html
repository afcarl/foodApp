Definitions

THINGS

Meal: a meal
Component: a part of the meal that consists of either ingredients or other components (e.g grilled chicken, sauteed onion and garlic)
Ingredient: a raw ingredient that makes up a component (e.g. garlic, raw chicken, )


RELATIONSHIPS

has: a meal 'has' components
includes: a component 'includes' other components or ingredients
